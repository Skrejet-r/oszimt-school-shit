
**Welche Informationen stellt der DHCP-Dienst (Dynamic Host Configuration Protocol) zur Verfügung?**

> [!INFO]- Lösung
> Folgende Informationen werden durch DHCP gestellt:
> - IP Adresse: wird eine eindeutige IP zugewiesen, um im Netzwerk zu odentifizieren
> - Subnetzmaske: definiert den Netzwerkbereich, zu dem die IP gehört
> - Standard-Gateway: IP Adresse des Routers, um Datenverkehr in andere Netzwerke weiterzuleiten
> - DNS-Server: IP Adressen des DNSs, die die Name auflösen
> - Lease Time: Gültigkeitsdauer der IP
