
**Welche Merkmale besitzt eine Europäische Gesellschaft (SE)?**

> [!INFO]- Lösung
>- Sie besitzt eine eigene Rechtspersönlichkeit 
>- Sie ist eine Kapitalgesellschaft
>- Das Mindestkapital beträgt 120000€
>- Jeder Aktionär haftet nur bis zur Höhe des von ihm gezeichneten Kapitals 
>- Die Geschäftsführung kann auf 2 Arten erfolgen: 
>	- Monistisches System 
>	- Dualistisches System
